# Contributing to rustlrn

Thank you for your interest in helping others learn Rust!

## How to Contribute

If you have a clear explanation for a Rust concept, feel free to submit a Pull Request or provide your notes.

## Recognition

We value every contribution. All contributors will be listed in the [README.md](README.md) file.
